# Electricity-Billing-System
Electricity Billing System | Java, Swing, MySQL

This project is a complete desktop application designed to automate electricity billing operations. The system provides separate Admin and User modules, enabling efficient billing management, secure payments, and real-time tracking of user data.

Key Features:

**Admin Module:**

- Add/delete/update customer connections

- Generate monthly bills based on units consumed

- View and manage all user records

**User Module:**

- View current bill

- Make online payments

- Check payment history

**Technical Highlights:**

- Java AWT/Swing GUI

- MySQL database integration

- Multithreading for smooth UI and processing

- Robust authentication system

- User-friendly forms and intuitive interface

**Tools Used:** NetBeans IDE, Java, JDBC, MySQL

This system demonstrates strong skills in GUI design, database connectivity, backend logic, and real-world software development workflows.
